js数据类型

2021-02-23 8 min read # js

通用知识：
HTML 标签类型：block、inline、inline-block、table……
JS中的数据类型：数字、字符串、布尔、函数、对象(obj、[]、{}、null)、未定义
方法： typeof 判断数据类型

例：

    var i = 100;
	alert( typeof i );			// number

###数据类型—Undefined

  • 使用var申明但是未初始化
  • typeof 返回Undefined
  • Undefined == null 为true
  • undefined 既不等于true也不等于false

数据类型—null

  • 申明变量应该赋值为 null
  • typeof 返回Object
  • null为0

数据类型—Boolean

  • 值为 true 和 false
  • 在Number()检测下 true =1； false=0；

例：

alert( Number(true) )   //1

数据类型—Number

1.转换方式

显式类型转换（强制类型转换）：
Number()
parseInt(对象 , 基数) a:整数不包含小数点 b:含有两个参数
W3C链接： http://www.w3school.com.cn/jsref/jsref_parseInt.asp
parseFloat(对象) a:浮点数，含小数点
W3C链接：http://www.w3school.com.cn/jsref/jsref_parseFloat.asp

隐式类型转换：
+ 200 + '3' 变成字符串
- * / % '200' - 3 变成数字
++ -- 变成数字
< 数字的比较 、字符串的比较
! 取反 把右边的数据类型转成布尔值
==

2.Number 超出范围

Infinity -Infinity 检测方式 isFinite( 对象 )

NAN
  • A: not a number
  • B: 非数值， NAN 不等于 NAN , 不等于任何数值
  • C：isNaN 函数用于检查其参数是否是非数字值
    含义： 是-不是一个数字
    用法： 可以转换为数值 返回 false。 反之 true

例：

var i = 9;       //数字
var b = 'ni';	  //字符

alert(isNaN(i))   // false
alert(isNaN(b))   // true

Math 对象方法 返回

abs(x) 绝对值
ceil(x) 上舍入
floor(x) 下舍入
max(x,y) 最高值
min(x,y) 最低值
random() 0 ~ 1 之间随机数
round(x) 四舍五入为接近的整数
valueOf() Math对象的原始值

数据类型—String

1.String.length方法，返回 字符长度。 从1 计数
2.String.charAt( 索引值 ) 从 0 计数

例：

var str = '字符测试';
alert( str.length );		//4
alert( str.charAt(2) );   //测
3.String.charCodeAt( 参数值 ) 返回字符的 Unicode 编码
  • 参数为空 — 返回 首字符Unicode
  • 0~9 48~57
    a~z 97~122
    A~Z 65~90
  • String.fromCharCode(22937, 21619) 根据 Unicode 编码返回对应字符
4.String.indexOf( 对象 )
  • 查找字符的位置， 找到：返回索引值 未找到：返回 -1；
  • String.indexOf(对象 ， 查找位置：数字) 默认从0，第二个参数可设置查找位置

例：

var str = 'm01iwniwwm';
for ( var i=0; i<str.length; i++ ) {
	if ( str.charAt(i) === 'i' ) {
		alert(i);
	}
}
// 返回 i 的位置  3 6

alert( str.indexOf('m') );                  //返回 0
alert( str.indexOf('m', 1) );               // 9
alert( str.indexOf('X') );					// -1 表示没找到
  • String.lastIndexOf(对象 ， 查找位置：数字)默认从最后一位，第二个参 数可设置查找位置， 如果第2个值为负数，当成0来处理
  • substring() 方法用于提取字符串中介于两个指定下标之间的字符。
    1. 出现负数 ， 当成 0 处理
    2. 可以自动检测 两个参数的大小 。 然后排序

例：

var str = '你是谁啊';
alert( str.substring(1,3) );	// 是谁
alert( str.substring(3,1) );	// 是谁
5.slice( 对象 )
  • slice()支持负数检测， 不会识别大小
  • alert( str.slice( 2, 0 ) ); 返回 空
    负数从后面倒着往前数~
    注： 倒数最后一位是-1
var str='abcdefg';
str.slice( 1, 2 ) //"b"
str.slice( 1, 3 ) //"bc"
str.slice( 1 ) //"bcdefg"
str.slice( 2, 0 ) //""
str.slice( -2 ) //"fg"
str.slice( -2,  str.lenght) //"fg"
6.转换大小写
  • alert( str.toUpperCase() ); // 转成大写
  • alert( str.toLowerCase() ); // 转成小写
7.split(字符 ， 数值-限定数组长度)方法用于把一个字符串分割成字符串数组。
  • 分割 返回数组

例：

var str = 'www.baidu.com';
alert( str.split('.') );				// [ 'www', 'baidu', 'com' ]
alert( str.split('.',2) );				// [ 'www', 'baidu' ]
alert( str.split('') );					// 每个字符都分割

var arr = str.split( '.' );
alert( arr[1] );                        // baidu
8.join()把数组转换为字符串 参数可以为空 空为 ','
  • 分割 返回数组

例：

var arr = [ 'aa', 'bb', 'cc' ];

alert( arr.join() );			//'aa'+','+'bb'+','+'cc'
alert( arr.join().length );     //8

alert( arr.join('') );          //aabbcc

alert( arr.join('-') );         // aa-bb-cc

7.像数组添加元素方法 a. Push() b. Unshift()

Push() 数组尾部新增，返回值为数组长度

例：

var arr = [ 1,2,3 ];

alert( arr.push( 'a' ) ); // 在数组尾部添加 a 返回新的数组长度 4

alert( arr ); // 1 2 3 a

unshift()数组头部新增，返回值为数组长度

例：

var arr = [ 1,2,3 ];
alert( arr.unshift( 0 ) );		

// 在数组头部添加 a 返回新的数组长度 4	IE 6  7 不支持 unshift 返回值

alert( arr );       //0 1 2 3
8.数组删除元素 a. pop() b.shift()

移除尾部 pop() 返回移除的对象

例：

var arr = [ '1', '2', '3', '4' ];
alert( arr.pop() );   //返回 4
alert( arr );          //  1 2 3

移除头部 shift() 返回移除对象

例：

var arr = [ '1', '2', '3', '4' ];
alert( arr.shift() );  // 返回 1
alert( arr );          // 返回 2 3 4
9.splice()方法向/从数组中添加/删除项目，然后返回被删除的项目
  1. 删除

例：

var arr = [ '1', '2', '3', '4' ];
        alert( arr.splice(0,1) );   
//返回 1  删除从0开始的第一个对象
        alert(arr)                  //  2 3 4

2.替换

例：

var arr = [ '1', '2', '3', '4' ];
        alert( arr.splice(0,2,'abc','345') );   
//返回 1 2 ,删除了0~2，新增了 abc 和 345
        alert(arr)                  //  abc 345 3 4

3.添加

例：

var arr = [ '1', '2', '3', '4' ];
        alert( arr.splice(1,0,'abc') );   
//返回空 , 索引1的位置新增了 abc
        alert(arr)                  //  1 abc 2 3 4
