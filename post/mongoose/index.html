MongoDB

MongoDB 是一个高性能，开源，无模式的文档型数据库。

[toc]

安装

安装方式分为**Homebrew安装和手动安装**

下载地址

在mac下安装mongodb

1. 将下载好的解压后文件放到 /usr/local 文件夹下。
2. 默认情况下， mongod进程使用该/data/db目录。

  • 在根目录/ 新建 data/db

3. 运行

  • 打开终端，切换到/usr/local/mongodb/bin 安装目录
  • 运行 ./mongod 启动服务端
  • 新开一个终端窗口，同样切换至安装目录，运行./mongo

4. 如下图所示，说明执行成功了

5. 将mongod二进制文件的位置添加至系统PATH变量
echo 'export PATH=/usr/local/mongodb/bin:$PATH'>>~/.bash_profile

  • 之后就可以直接使用 mongod ， mongo

6. 自定义数据库路径
如果不使用默认数据目录/data/db，请使用--dbpath自定义数据库路径

mongod --dbpath <数据库完整路径>

数据库使用

文档地址

基础命令

  • 查看所有数据库
    show dbs

  • 使用数据库、创建数据库
    use test

  • 插入一条数据
    db.test.insert({'name':'yue'});

  • 查找刚才插入的数据
    db.test.find()

  • 删除数据库
    db.dropDatabase()

常用的增删改查

增

创建集合
db.createColletion('cs')
创建一个cs的集合

插入一条数据
db.test.insert({'name':'yue'});

导入数据
mongoimport --db test --collection cs --drop --file primer-dataset.json
在test数据库下的cs集合里面导入数据，清空之前的数据

删

删除数据库
db.dropDatabase()

删除集合
db.test.drop()

删除所有集合
db.test.remove({})

删除所有num为11的数据
db.test.remove({'age':11})

如果只想删除找到的第一条
db.test.remove({'age':11}, { justOne: true } )

改

修改数据，有前置条件，查询

db.test.update({'age':11},{$set:{'age':22}})
注：test集合下寻找到第一个符合条件（age为11）的数据，将它改成age:22

修改所有符合条件的数据
db.test.update({'age':11},{$set:{'age':22}},{multi:true})

完整替换
db.test.update({'age':22},{'age':11})

查

寻找所有数据
db.test.find()

精确查找
db.test.find({'age':11})
注：查找所有age等于11的数据

多个条件查找
db.test.find({'name':'3','age':112})

带运算的查找
db.test.find({'age':{$gt:11}})
注：寻找所有age大于11的数据

  • 排列方式-升序排列
    db.test.find({'age':{$gt:11}}).sort({"age":1})
  • -1将序排列

Limit与Skip方法
limit(num)读取指定数量(num)的数据记录
skip(num)跳过指定数量(num)的数据
结合在一起可以处理分页数据
db.cs.find({},{"age":1}).limit(10).skip(10)
注：读取cs集合下，11-20条数据

符号 用法
$lt 小于号
$gt 大于号
$gte 大于等于
$lte 小于等于
